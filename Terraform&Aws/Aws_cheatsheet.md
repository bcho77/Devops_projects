# AWS Services Cheatsheet (DevOps & Cloud)

## Compute

* **EC2** – Virtual servers in the cloud
* **Auto Scaling** – Automatically scale EC2 instances
* **Elastic Beanstalk** – PaaS for app deployment
* **Lambda** – Serverless functions
* **Lightsail** – Simplified VPS
* **ECS** – Container orchestration (AWS-native)
* **EKS** – Kubernetes on AWS
* **Batch** – Batch computing jobs

---

## Storage

* **S3** – Object storage
* **S3 Glacier** – Archive storage
* **EBS** – Block storage for EC2
* **EFS** – Network file system
* **FSx** – Managed file systems (Windows/Lustre)
* **Storage Gateway** – Hybrid storage

---

## Databases

* **RDS** – Managed relational databases
* **Aurora** – High-performance MySQL/PostgreSQL
* **DynamoDB** – NoSQL key-value database
* **ElastiCache** – Redis / Memcached
* **Redshift** – Data warehouse
* **Neptune** – Graph database
* **DocumentDB** – MongoDB-compatible

---

## Networking & Content Delivery

* **VPC** – Virtual private cloud
* **Subnets** – Network segmentation
* **Internet Gateway** – Internet access
* **NAT Gateway** – Outbound internet for private subnets
* **Route 53** – DNS & domain management
* **Elastic Load Balancer (ALB/NLB/CLB)**
* **CloudFront** – CDN
* **API Gateway** – API management

---

## Security, Identity & Compliance

* **IAM** – Identity & access management
* **IAM Roles** – Secure permissions
* **Cognito** – User authentication
* **KMS** – Encryption key management
* **Secrets Manager** – Secrets storage
* **ACM** – SSL/TLS certificates
* **WAF** – Web application firewall
* **Shield** – DDoS protection

---

## Monitoring & Management

* **CloudWatch** – Logs, metrics, alarms
* **CloudTrail** – API auditing
* **AWS Config** – Resource compliance
* **Systems Manager** – Ops management
* **Trusted Advisor** – Best practices
* **Health Dashboard** – Service health

---

## DevOps & CI/CD

* **CodeCommit** – Git repositories
* **CodeBuild** – Build service
* **CodeDeploy** – Deployment automation
* **CodePipeline** – CI/CD pipelines
* **CloudFormation** – Infrastructure as Code
* **CDK** – IaC with programming languages
* **Amplify** – Frontend & full-stack hosting

---

## Analytics & Big Data

* **Athena** – Query S3 using SQL
* **Glue** – ETL service
* **EMR** – Hadoop/Spark
* **Kinesis** – Real-time streaming
* **OpenSearch** – Search & analytics
* **QuickSight** – BI dashboards

---

## Application Integration

* **SQS** – Message queues
* **SNS** – Pub/Sub messaging
* **EventBridge** – Event bus
* **Step Functions** – Workflow orchestration
* **AppSync** – GraphQL APIs

---

## Migration & Hybrid

* **Migration Hub** – Migration tracking
* **DMS** – Database migration
* **DataSync** – Data transfer
* **Snowball** – Offline data transfer
* **Outposts** – AWS on-prem

---

## Machine Learning & AI

* **SageMaker** – ML development
* **Rekognition** – Image/video analysis
* **Comprehend** – NLP
* **Textract** – Document analysis
* **Polly** – Text-to-speech

---

## Cost Management

* **Cost Explorer** – Cost analysis
* **Budgets** – Cost alerts
* **Savings Plans** – Cost optimization
* **Reserved Instances** – EC2 discounts
